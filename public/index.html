Projects This section will showcase my personal projects outside of work

Project 1:  Website V1.0

Automated integration and deployment of my personal website using Jenkins, AWS, Dockers,and SonarQube.

The project code is hosted on GitHub, with Jenkins, Docker, and SonarQube deployed on different AWS EC2 instances. Whenever a new commit is pushed, Jenkins triggers SonarQube to check the code, then deploys the code to the server running Docker. The Dockerfile is then used to create a new image, and Nginx is used to serve the frontend pages. Finally, AWS Route 53 is used to resolve my personal domain, linking it to my personal website.

Project Details

Project 2:  Website V2.0(This Website)

Automated integration and deployment of my personal website using Github Actions.

Since the services for Jenkins, Docker, and SonarQube require processors with sufficient computing power and storage space, and we might add more services to the process in the future, the free AWS EC2 instances do not fully meet these requirements. Therefore, we use GitHub Actions to handle the CI/CD part of the work, and use the free AWS EC2 instances only to run the Nginx service to host the website, thereby reducing costs.
